# 快速开始 🚀

欢迎使用番茄土豆！按照以下步骤快速开始使用。

## 5分钟快速上手

### 步骤1：安装依赖（首次使用）

```bash
npm install
```

### 步骤2：启动开发服务器

```bash
npm run dev -- --host 127.0.0.1
```

### 步骤3：打开浏览器

访问：http://127.0.0.1:5173

## 第一次使用

### 1. 创建你的第一个任务

1. 在首页输入框中输入任务标题，例如："完成项目报告"
2. 添加标签（可选），例如："工作,重要"
3. 设置预计番茄数，例如：3
4. 点击"添加任务"

### 2. 开始你的第一个番茄钟

1. 点击任务卡片上的"开始"按钮
2. 番茄计时器会出现在页面顶部
3. 点击"开始"按钮开始计时
4. 专注工作25分钟

### 3. 查看统计数据

1. 点击左侧导航栏的"数据统计"
2. 查看你的番茄完成情况
3. 分析时间分配

## 常用命令

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview

# 代码检查
npm run lint
```

## 推荐配置

### 1. 设置番茄时长

1. 点击左侧导航栏的"设置"
2. 调整番茄时长（建议25分钟）
3. 设置短休息（建议5分钟）
4. 设置长休息（建议15分钟）

### 2. 配置GitHub同步（可选）

如果你想在多个设备间同步数据：

1. 在GitHub创建一个仓库
2. 生成Personal Access Token
3. 在设置页面配置GitHub同步
4. 点击"上传到GitHub"备份数据

详细步骤请查看 [USER_GUIDE.md](./USER_GUIDE.md#6-github同步配置)

## 使用技巧

### 💡 提高效率
- 每天开始前规划任务
- 合理估算番茄数
- 定期查看统计数据
- 使用标签分类任务

### 🎯 保持专注
- 番茄钟期间关闭通知
- 准备好所需材料
- 避免切换任务
- 完成后休息

### 💾 数据安全
- 定期导出数据备份
- 配置GitHub同步
- 不要清除浏览器数据
