# 番茄土豆网页工具需求文档

## 1. 工具概述

### 1.1 工具名称

番茄土豆网页工具

### 1.2 工具描述

一款高效的番茄钟任务管理工具，结合番茄工作法和任务追踪功能，帮助用户提升工作效率和时间管理能力。

## 2. 核心功能

### 2.1 主界面布局

- **左侧区域**：显示开始番茄按钮和最新的番茄记录，显示开始时间与结束时间
- **右侧区域**：顶部为任务输入文本框，支持通过#xx实现标签区分，下侧为任务列表，支持拖拽更改位置，双击编辑内容
- **底部区域**：统计、番茄历史、任务历史信息展示

### 2.2 任务管理

- **任务创建**：通过右侧文本输入框输入任务，支持#标签分类
- **任务操作**：任务列表支持拖拽排序，双击编辑任务内容
- **任务设置**：每个任务侧边栏支持弹出窗口设置提醒时间和提醒内容
- **任务完成**：已完成的番茄任务自动从列表中删除
- **任务描述修改**：每个番茄完成后，在开始番茄部分可修改当前番茄的任务描述

### 2.3 番茄时间设置

- 支持自定义番茄时间，默认为25分钟
- 用户可根据需要调整单个番茄时间段长度
- 实时显示番茄开始时间和预计结束时间

### 2.4 数据统计与可视化

- **统计面板**：显示当日累计番茄数量和任务完成情况
- **番茄历史**：累计完成番茄数量统计
- **任务历史**：累计完成任务数量统计，支持趋势图表展示

### 2.5 历史记录管理

- **历史番茄记录**：支持查看、修改、添加、删除历史番茄记录
- **历史任务管理**：按时间顺序记录所有番茄任务，支持查看历史任务完成情况
- **数据恢复**：支持恢复删除的任务记录

### 2.6 数据同步与备份

- 依托GitHub API接口实现数据同步
- 支持将历史番茄数据保存到GitHub仓库
- 支持从GitHub仓库拉取数据到当前界面
- 本地数据与远程仓库的同步机制
- 需要设计合理的数据保存结构

### 2.7 服务部署

- 支持用GitHub Page的形式部署

## 3. 设计风格

### 3.1 配色方案

主色调采用温暖的番茄红（#FF6B6B）搭配清新的薄荷绿（#4ECDC4），辅以中性灰白色背景

### 3.2 视觉细节

圆角设计（8px圆角）营造亲和感，卡片式布局带有轻微阴影效果，图标采用线性风格保持简洁

### 3.3 整体布局

采用左右分栏布局，左侧显示番茄控制和记录，右侧为任务管理和统计信息，界面简洁明了便于操作
