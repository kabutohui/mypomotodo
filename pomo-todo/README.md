# PomoTodo - 番茄土豆时钟

PomoTodo 是一个结合了番茄工作法和待办事项管理的网页应用，帮助你提高工作效率和时间管理能力。

## 功能特点

### 番茄时钟
- 支持自定义番茄时间（默认25分钟）
- 可视化倒计时显示
- 完成提醒功能

### 待办事项管理
- 添加、编辑、删除待办事项
- 支持拖拽调整任务顺序
- 任务置顶功能
- 为任务计时专注时间

### 标签系统
- 支持以 `#标签` 形式为任务添加标签
- 按标签统计专注时间和番茄数量

### 数据同步
- 集成代码仓库同步功能
- 支持虚拟存储和本地存储两种方式
- 自动保存和同步数据

### 统计数据
- 显示每日完成的番茄数量和专注时间
- 按标签统计工作分布
- 任务历史记录

## 如何使用

1. 打开 `index.html` 文件启动应用
2. 添加待办事项（支持 `#标签` 格式）
3. 选择一个任务，点击时钟按钮开始专注
4. 完成番茄时间后，系统会自动记录并更新统计数据
5. 配置同步功能以备份数据

## 数据存储

应用支持两种存储方式：
- 虚拟存储：数据存储在浏览器本地
- 本地存储：数据存储在你的计算机文件系统中（需要浏览器权限）
