使用帮助

番茄时钟

设置专注时间（默认25分钟），点击开始按钮启动计时器。完成一个番茄后，可以休息5分钟再开始下一个。

待办事项

  • 点击"添加任务"创建新的待办事项
  • 任务支持标签，使用 #标签名 格式
  • 可以拖动任务上下移动以调整顺序
  • 点击任务上的星标按钮可以将任务置顶
  • 点击任务上的时钟按钮可以开始专注

GitHub 同步

点击右上角的 GitHub 图标配置同步信息，可以将任务数据同步到 GitHub 仓库中。支持虚拟存储和本地存储两种方式。

统计数据

应用会记录每日完成的番茄数量和专注时间，并按标签进行统计，帮助你了解自己的工作习惯。
